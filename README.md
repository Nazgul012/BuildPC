# BuildPC
BuildPC — онлайн-платформа та сервіс для підбору, конфігурування та замовлення комплектуючих для складання персонального комп’ютера. Користувач може обрати мету (ігри, робоча станція, монтаж відео тощо), вказати бюджет і отримати готові збірки, сумісні компоненти, докладні інструкції зі складання та рекомендації щодо апгрейду.
